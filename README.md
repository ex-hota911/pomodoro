# App

```
                 +-------+
                 | Init  |
                 +---+---+
                     |
                     | +-----+
                     | |     |
                 +---v-v-+   | Reset
   Stop/Resume   |       +---+
            +----> Work  |
+------+    |    |       <---------------+
|      |    |    +---+---+               |
| Stop <----+        |                   |
|      |    |        | after 25 min      | After 5 min
+------+    |        |                   | or Reset
            |    +---v---+               |
            +----> Break +---------------+
                 +-------+
````
